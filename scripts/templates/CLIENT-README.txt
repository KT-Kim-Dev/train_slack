Intra-Chat 클라이언트 (Windows portable exe)
============================================

설치 없이 exe 파일 하나만 실행하면 됩니다.

## 사용 방법

1. Intra-Chat-X.X.X-portable.exe 를 더블클릭
2. 관리자가 발급한 ID/PW 로 로그인

## 주의

- exe 빌드 시 서버 IP가 내장됩니다.
- 서버 IP가 바뀌면 클라이언트 exe 를 다시 빌드해야 합니다.
- Windows Defender SmartScreen 경고가 뜰 수 있습니다 (코드사인 미적용 PoC).

## 서버 연결 확인

서버 PC에서 start-server.bat 이 실행 중이어야 합니다.
기본 포트: 3000
